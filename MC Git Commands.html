How to Use GitHub

Follow these steps to Add Files:

Git Installation

Install Git Core from Link Below:

Git Core App for Windows

Create a Repository

In your GitHub account create New Repository

The upper link, is the Link to this repository

File Creation

You can create a File in any Text Editor

Git Commands

You have to enter the following commands in terminal of VS Code

git init (will initialize GIT)

git add "fileName" (will add file to repo)

git clone "link to your repo" (Will copy your data from repo link)

git commit -m "Your message" (will commit your changes)

git push (will push/upload your file to online repo server)
